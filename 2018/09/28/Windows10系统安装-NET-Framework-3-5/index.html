Windows10系统安装 .NET Framework 3.5

发表于 2018-09-28

1、获取压缩包
点击链接获取： https://pan.baidu.com/s/1wfOA3lnOcr9e3TCE-61Zfg
密码：ytax

2、解压包
将下载的NET Framework 3.5.zip解压，解压后如下图

3、移动文件
将解压得到【sxs】文件夹复制到C盘根目录（用完可以删除）

4、运行
右键> 以管理员身份运行> NET Framework 3.5，完成

温馨提示：一定要【右键】以管理员身份运行NET Framework 3.5

5、等待完成
随后会出现一个黑框，等到100%就完成了！有时速度会很快，闪一下就过去了
